# Простая RPG игра

Написана на **nasm** под **Linux**.

### Сборка

В любой папке прописать следующие команды:

```
git clone https://github.com/Wallxxx/Assembler.git
cd Assembler/
git checkout RPG
make all file=main
```

Запуск: ./main
